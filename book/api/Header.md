`Header` (component)
====================

A header toolbar for holding buttons

```xml
<Header  />
```

Properties
----------

### `children`

Child nodes

type: `node`


### `leftMenuItems`

Array of menu items to add to the header

type: `node`



### `style`

Style config.

type: `object`


### `title`

Title of the Header

type: `string`
