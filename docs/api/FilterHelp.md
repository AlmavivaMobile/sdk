`FilterHelp` (component)
========================

Help dialog for filter fields.

```xml
<FilterHelp />
```

Properties
----------


### `monoStyle`

Mono style config.

type: `object`
defaultValue: `{
  fontWeight: 'bold',
  display: 'inline-block',
  minWidth: '80px'
}`


### `style`

Style config.

type: `object`
defaultValue: `{
  padding: '12px',
  margin: '36px',
  maxWidth: '600px'
}`
