`Button` (component)
====================

Button with built-in tooltip.

Properties
----------

### `action`

Should we display as a floating action button?

type: `bool`


### `className`

Css class name to apply on the span.

type: `string`


### `disabled`

Should this button be disabled?

type: `bool`


### `tooltip`

The tooltip to show for this button.

type: `string`


### `tooltipStyle`

Style for tooltip element.

type: `object`
