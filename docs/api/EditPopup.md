`EditPopup` (component)
=======================

Popup that can be used for feature editing (attribute form).

```xml
<EditPopup map={map} />
```

Properties
----------

### `className`

Css class name to apply on the root element of this component.

type: `string`



### `map` (required)

The ol3 map to register for singleClick.

type: `instanceOf ol.Map`
